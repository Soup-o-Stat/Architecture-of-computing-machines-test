## Блок вопросов 1
### 1. Вычислительная машина - это...
Вычислительная машина - это устройство для выполнения арифметических и логических операций.
### 2. Вычислительная система - это...
Вычислительная система - это совокупность вычислительных машин и программного обеспечения для обработки данных
### 3. Электронная вычислительная машина (ЭВМ) - это...
Электронная вычислительная машина (ЭВМ) – вычислительная машина, использующая электронные компоненты
### 4. Архитектура вычислительной машины - это...
Архитектура вычислительной машины – описание структуры, принципов работы и взаимодействия компонентов вычислительной системы
### 5. Функциональная организация ВМ - это...
Функциональная организация ВМ – распределение функций между узлами и модулями ВМ
### 6. Структурная организация ВМ - это...
Структурная организация ВМ – физическое размещение и соединение компонентов вычислительной машины
### 7. Какой вклад в развитие вычислительной техники внес Чарльз Бэбидж?
Чарльз Бэбидж разработал концепцию аналитической машины – прототипа современного компьютера
### 8. Для чего были изобретены перфокарты в начале 19 века?
Перфокарты изобретены для автоматизации обработки данных (ткачество, вычисления)
### 9. К первому поколению по элементной базе относятся какие компьютеры?
Компьютеры на электронных лампах.
### 10. Ко второму поколению по элементной базе относятся какие компьютеры?
Компьютеры на транзисторах.
### 11. К третьему поколению по элементной базе относятся какие компьютеры?
Компьютеры на интегральных схемах.
### 12. К четвертому поколению по элементной базе относятся какие компьютеры?
Компьютеры на микропроцессорах.
### 13. К пятому поколению по элементной базе относятся какие компьютеры?
Компьютеры с элементами искусственного интеллекта.
### 14. Какие устройства, согласно принципам фон Неймана, должны входить в состав компьютера?
Устройства по фон Нейману: процессор, память, устройство ввода-вывода.
### 15. Назовите классы вычислительных машин по принципу действия
Аналоговые, цифровые, гибридные.
### 16. Назовите классы вычислительных машин по назначению.
Универсальные, специализированные, суперкомпьютеры.
### 17. Назначение дешифратора
Дешифратор – устройство для преобразования кода в один активный сигнал.
### 18. Назначение мультиплексора.
Мультиплексор – коммутирует несколько входных сигналов в один выходной.
### 19. Назначение шифратора.
Шифратор – преобразует активный сигнал в код.
### 20. Назначение приоритетного шифратора
Приоритетный шифратор – кодирует сигнал с наивысшим приоритетом.
### 21. Что такое формат команды?
Формат команды – структура машинной команды, определяющая её длину и расположение частей (операции и адресов)
### 22. Какая команда называется четырехадресной?
Четырехадресная команда – команда, содержащая адреса двух операндов, результата и следующей команды
### 23. Какой регистр в процессоре с трехадресным форматом команды содержит адрес следующей выполняемой команды?
В трехадресном формате команд адрес следующей команды хранится в счетчике команд.
### 24. Где находится адрес для хранения результата операции в двухадресной команде?
В двухадресной команде адрес результата совпадает с одним из адресов операндов.
### 25. Для реализации ВМ с одноадресным форматом команд специально введен какой регистр?
Для одноадресного формата используется аккумулятор как регистр хранения промежуточных результатов.
### 26. Стековая ВМ использует какие форматы команд?
Стековая ВМ использует команды безадресного формата.
### 27. Что является основой ЭВМ?
Основой ЭВМ является процессор, память и устройства ввода-вывода.
### 28. Почему основной процессор в ВМ называется центральным?
Центральный процессор называется так, поскольку управляет всей вычислительной системой.
### 29. Чем отличаются фоннеймановские ВМ и ВМ гарвардского типа?
Фоннеймановская архитектура использует общую память для данных и команд, гарвардская – раздельную.
### 30. Какая архитектура ВМ получила наибольшее распространение?
Наибольшее распространение получила фоннеймановская архитектура.
### 31. Какая архитектура считается фоннеймановской?
Фоннеймановская архитектура – общая память для команд и данных.
### 32. Какие основные части включает в себя архитектура машины фон Неймана?
Основные части машины фон Неймана: процессор, память, устройства ввода-вывода.
### 33. Как осуществляется взаимодействие устройств машины при выполнении на ней машинных операций (на примере операции сложения на одноадресной машине)?
На одноадресной машине операция сложения выполняется через загрузку операндов в аккумулятор, выполнение операции и запись результата в память.
### 34. Принципы фон Неймана.
Принципы фон Неймана: линейное программирование, хранение программ в памяти, последовательное выполнение команд.
### 35. Базовая аппаратная конфигурация персонального компьютера. Системный блок. Основные узлы.
Системный блок, материнская плата, процессор, оперативная память, дисковая память, видеокарта.
### 36. Материнская плата. Основные узлы. Процессор.
Материнская плата: содержит процессорный разъем, оперативную память, чипсет, шины и разъемы для периферии.
### 37. Каковы основные устройства ВМ?
Основные устройства ВМ: процессор, оперативная память, дисковая память, устройства ввода-вывода.
### 38. Что такое процессор ВМ?
Процессор ВМ – основной узел, выполняющий обработку данных и управление работой системы.
### 39. Какая функция не относится к самому нижнему уровню управления центрального процессора?
Функции высокого уровня, например, интерпретация команд, не относятся к нижнему уровню управления процессором.
### 40. Какую функцию в операционном устройстве центрального процессора выполняет мультиплексор?
Мультиплексор в процессоре выбирает один из нескольких входных сигналов для передачи на выход.
### 41. Какую функцию в операционном устройстве центрального процессора выполняет дешифратор?
Дешифратор преобразует машинный код команды в управляющие сигналы для выполнения операции.
### 42. Какие типы устройства управления ЦП выделяют по способам формирования управляющего слова ВМ?
Типы устройств управления ЦП: на жесткой логике и на микропрограммной логике.
### 43. Какой цифровой блок имеет обозначение DC?
DC (Decoder Circuit) – дешифратор.
### 44. Какой цифровой блок имеет обозначение MS?
MS (Multiplexer Selector) – мультиплексор.
### 45. Что такое память ВМ? Перечислите различные виды памяти по функциональному назначению.
Память ВМ – устройство для хранения данных и программ. Виды: оперативная, постоянная, кэш, внешняя, регистровая, виртуальная.
### 46. Какой цифровой блок имеет обозначение SRAM?
SRAM – статическая оперативная память.
### 47. Какой цифровой блок имеет обозначение RAM?
RAM – оперативная память с произвольным доступом.
### 48. Какой цифровой блок имеет обозначение DRAM?
DRAM – динамическая оперативная память.
### 49. Какой цифровой блок имеет обозначение DDR RAM?
DDR RAM – память с удвоенной скоростью передачи данных.
### 50. Какой цифровой блок имеет обозначение ROM?
ROM – постоянная память.
### 51. Закончите определение: «Часть памяти компьютера, которая используется для хранения программ и данных во время решения задачи, называется …».
оперативной памятью.
### 52. Назовите вид памяти, в которой возможен доступ ко многим ячейкам памяти по их содержимому
Ассоциативная память.
### 53. В каком типе памяти данные организуются в виде некоторой файловой системы?
Внешняя память (например, жесткие диски, SSD).
### 54. Как называется интервал времени от момента посылки запроса информации до момента получения результата на шине данных?
Время доступа.
### 55. Что означает: «эффективность кэш-памяти равна 1»?
Эффективность кэш-памяти равна 1, если все обращения происходят к данным из кэша.
### 56. Стековая память – это (дайте определение).
Стековая память – память с доступом по принципу LIFO (последний вошел – первый вышел).
### 57. На что указывает указатель стека?
Указатель стека указывает на вершину стека.
### 58. Для чего используется кэш-память?
Кэш-память используется для ускорения доступа к часто используемым данным.
### 59. Какая память в ЭВМ является наиболее быстродействующей?
Регистры процессора – самая быстродействующая память.
### 60. Для чего предназначен цикл прерывания в устройстве управления центрального процессора?
Цикл прерывания обеспечивает обработку внешних или внутренних событий.
### 61. Для чего применяется Польская Инверсная (бесскобочная) Запись (ПОЛИЗ)?
ПОЛИЗ используется для упрощения вычислений и оптимизации работы процессора.
### 62. Назовите правильное окончание определения: «Часть памяти компьютера, которая используется для долговременного хранения программ и данных, называется …».
внешней памятью.
### 63. Укажите тип памяти, к которому относится следующее описание: «Данные хранятся на полупроводниковом кристалле, их можно читать и записывать только блоками, чтением и записью управляет контроллер».
Flash-память.
### 64. Что означает английское сокращение SSD?
SSD – Solid State Drive (твердотельный накопитель).
### 65. В каком виде вводится в ЭВМ текст с помощью сканера?
Текст вводится в виде изображения.
### 66. Какие устройства используются для ввода графической информации в ЭВМ?
Устройства для ввода графической информации: сканеры, графические планшеты, камеры.
### 67. Какие достоинства характерны для порта USB?
Достоинства USB: универсальность, высокая скорость передачи данных, поддержка горячего подключения.
### 68. Какие характеристики относятся к накопителям с жесткими магнитными дисками
Характеристики жестких дисков: емкость, скорость вращения шпинделя, время доступа, интерфейс подключения.
### 69. Форм-фактор накопителя на жестком магнитном диске – это (дайте определение).
Форм-фактор жесткого диска – его физический размер и совместимость с устройствами.
### 70. Назначение регистра – счетчика команд.
Регистр-счетчик команд хранит адрес следующей команды, подлежащей выполнению.

## Блок вопросов 2
### 1. Порт ввода-вывода – это (дайте определение)
Порт ввода-вывода – интерфейс, обеспечивающий передачу данных между процессором и внешними устройствами.
### 2. Какие шины входят в состав общей шины (системной магистрали)?
В состав общей шины входят шины данных, адреса и управления.
### 3. Что относится к периферийным устройствам ВМ?
К периферийным устройствам относятся принтеры, сканеры, мониторы, клавиатуры, мыши, накопители.
### 4. Дайте определение интерфейсу ВМ
Интерфейс ВМ – набор аппаратных и программных средств для взаимодействия между устройствами и пользователем.
### 5. Какие программы обычно хранятся в ПЗУ современных персональных компьютеров?
В ПЗУ современных ПК хранятся программы BIOS/UEFI, загрузчики и диагностические утилиты.
### 6. Операционная система – это (дайте определение)
Операционная система – программное обеспечение, управляющее аппаратными ресурсами и предоставляющее интерфейс для приложений и пользователей.
### 7. Прикладное программное обеспечение – это (дайте определение).
Прикладное ПО – программы, выполняющие задачи пользователя (например, текстовые редакторы, браузеры).
### 8. Разделение времени (многопользовательский режим) для операционной системы – это (дайте определение).
Разделение времени – способ работы ОС, при котором ресурсы процессора динамически распределяются между несколькими пользователями или задачами.
### 9. Пакетная обработка – это (дайте определение).
Пакетная обработка – выполнение нескольких задач по заранее сформированному списку без вмешательства пользователя.
### 10. Мобильная (переносимая) ОС – это (дайте определение)
Мобильная ОС – операционная система, способная работать на устройствах с разной аппаратной платформой.
### 11. Фрагментация памяти – это (дайте определение).
Фрагментация памяти – неравномерное распределение памяти, при котором доступные области памяти разобщены.
### 12. Операционные системы реального времени – это (дайте определение)
ОС реального времени – операционные системы, способные обрабатывать задачи в строго заданные временные интервалы.
### 13. Симметричные ОС – это (дайте определение).
Симметричные ОС – операционные системы, поддерживающие многопроцессорные конфигурации с равноправным распределением задач.
### 14. Какие достоинства имеет монолитная архитектура ОС?
Достоинства монолитной архитектуры ОС: высокая производительность, прямое взаимодействие компонентов.
### 15. Какие основные средства применяются для синхронизации процессов в системах с общей памятью?
Средства синхронизации процессов: семафоры, мьютексы, блокировки, очереди сообщений, условные переменные.
### 16. Какое основное средство используется для синхронизации процессов в распределенных системах с разделенной памятью?
Для синхронизации процессов в распределенных системах используется распределенный мьютекс.
### 17. Файловая система – это (дайте определение)
Файловая система – способ организации, хранения и управления данными на носителях информации.
### 18. Специальные файлы - это (дайте определение)
Специальные файлы – виртуальные файлы, представляющие устройства или процессы (например, /dev/null в Linux).
### 19. Физическая структура файла - это (дайте определение)
Физическая структура файла – размещение данных на носителе (последовательный, индексный, кластерный).
### 20. Оперативная память. Постоянная память
Оперативная память – временная память для быстрого доступа; постоянная память – энергонезависимая память для длительного хранения данных.
### 21. Внешняя память.
Внешняя память – устройства для долговременного хранения данных (жесткие диски, SSD, флешки).
### 22. Системные программы. Служебные программные средства. Назначение и примеры.
Системные программы – обеспечивают управление ресурсами (например, ОС). Служебные средства – утилиты для обслуживания системы (антивирусы, архиваторы).
### 23. Понятие вычислительного процесса.
Вычислительный процесс – выполнение последовательности операций над данными для решения задачи.
### 24. Состояние процесса.
Состояние процесса – текущий статус процесса (активный, готовый, ожидающий).
### 25. Понятие потока. Многопоточность
Поток – минимальная единица выполнения в процессе. Многопоточность – способность программы выполнять несколько потоков параллельно.
### 26. Понятие ресурса. Распределение и управление ресурсами
Ресурс – объект, доступный для использования процессами. Управление включает распределение, блокировку и освобождение ресурсов.
### 27. Многозадачность (вытесняющая и невытесняющая).
Вытесняющая – процессор принудительно переключается между задачами.
Невытесняющая – задача сама уступает процессор.
### 28. Прерывания. Механизм прерываний.
Прерывания – события, приостанавливающие выполнение программы для обработки внешнего или системного сигнала. Механизм: срабатывание триггера, вызов обработчика.
### 29. Классификация операционных систем. Особенности областей использования (системы пакетной обработки, системы разделения времени, системы реального времени)
Системы пакетной обработки – последовательная обработка задач.
Системы разделения времени – многопользовательские, с динамическим распределением ресурсов.
Системы реального времени – обработка событий с жесткими временными ограничениями.
### 30. Архитектура операционных систем. Понятие ядра операционной системы. Механизмы работы ядра. Структура ядра операционной системы
Архитектура ОС – структура взаимодействия компонентов ОС. Ядро ОС – центральная часть, управляющая ресурсами и взаимодействием аппаратуры и программ.
### 31. Вспомогательные модули операционной системы.
Вспомогательные модули ОС – компоненты, выполняющие дополнительные функции 
### 32. Концепция микроядерной архитектуры.
Микроядерная архитектура – концепция ОС, где ядро минимально, а большинство функций выполняется в пользовательском пространстве.
### 33. Файловые системы. Основные понятия и функции.
Файловая система – структура для хранения данных на носителях, обеспечивающая доступ, управление, защиту файлов.
### 34. Файлы. Доступ к файлам.
Файлы – именованные области памяти; доступ осуществляется через пути, индексы или дескрипторы.
### 35. Какие имеются методы управления вводом/выводом?
Методы управления вводом/выводом: программный, прерывания, прямой доступ к памяти (DMA).
### 36. В чем состоит конвейеризация вычислений?
Конвейеризация вычислений – выполнение операций параллельно за счет разбиения их на этапы.
### 37. Что такое суперскалярные процессоры?
Суперскалярные процессоры – процессоры, способные выполнять несколько инструкций за такт.
### 38. В чем состоит гиперпотоковая обработка?
Гиперпотоковая обработка (Hyper-threading) – технология, позволяющая одному ядру обрабатывать несколько потоков.
### 39. Дайте определение процессорам с архитектурой CISC, RISC, VLIW, EPIC
CISC – сложные команды; RISC – минималистичные инструкции; VLIW – длинные инструкции; EPIC – высокая параллельность.
### 40. Что такое параллелизм в программах?
Параллелизм – выполнение нескольких задач одновременно для повышения производительности.
### 41. Что представляет классификация ВС по Флинну?
Классификация Флинна – SISD, SIMD, MISD, MIMD; различия в способе обработки потоков данных и инструкций.
### 42. Чем характеризуется архитектура памяти ВС?
Архитектура памяти – организация доступа к памяти (разделяемая, распределённая, кэш-память).
### 43. Чем различаются физически разделяемая память и распределённая память?
Разделяемая память – общая для всех процессоров; распределённая – локальная для каждого.
### 44. Охарактеризуйте статические и динамические топологии ВС.
Статические топологии – фиксированные структуры (кольцо, звезда); динамические – изменяемые по запросу
### 45. Чем характеризуются решетчатые топологии и топология гиперкуба?
Решетчатая топология – упорядоченные соединения; гиперкуб – многомерное соединение.
### 46. Что такое векторные и матричные вычислительные системы класса SIMD?
SIMD – системы, обрабатывающие массивы данных (векторные, матричные вычисления).
### 47. Охарактеризуйте MIMD–системы с разделяемой памятью.
MIMD с разделяемой памятью – параллельный доступ к общей памяти.
### 48. Охарактеризуйте MIMD–системы с распределённой памятью.
MIMD с распределённой памятью – независимые узлы с локальной памятью.
### 49. Каковы основные тенденции развития высокопроизводительных ВМ и ВС?
Основные тенденции: увеличение производительности, снижение энергопотребления, развитие квантовых вычислений.
### 50. Назовите основные логические (булевы) функции.
Логические функции: И, ИЛИ, НЕ, И-НЕ, ИЛИ-НЕ, XOR.
### 51. Для каждой из функций И, ИЛИ, НЕ запишите таблицу истинности и изобразите элементы их реализующие.
И: 1, если оба 1.
ИЛИ: 1, если хотя бы один 1.
НЕ: инверсия значения.
### 52. Для каждой из функций И-НЕ, ИЛИ-НЕ, запишите таблицу истинности и изобразите элементы их реализующие.
И-НЕ и ИЛИ-НЕ: дополнение функций И и ИЛИ.
### 53. Для функции ИСКЛЮЧАЮЩЕЕ ИЛИ запишите таблицу истинности и изобразите элемент её реализующий.
XOR: 1, если входы различны.
### 54. Что такое язык описания аппаратуры? Назовите существующие языки описания аппаратуры, в чем их отличие?
Язык описания аппаратуры – языки для проектирования схем (VHDL, Verilog). Отличия: синтаксис, уровень абстракции.
### 55. Из чего состоит управляющее слово операционного устройства центрального процессора?
Управляющее слово – набор бит, определяющий управление устройствами процессора.
### 56. Какие управляющие входы имеет арифметико-логическое устройство учебной ВМ?
Управляющие входы АЛУ: выбор операции, ввод/вывод данных, сброс.
### 57. Если в ВМ применяется только один метод адресации операндов, то какой цикл устройства управления не используется?
Адресный цикл не используется при непосредственной адресации.
### 58. Чем определяется номер цикла команды в устройстве управления учебной ВМ?
Номер цикла команды определяется шагами выполнения инструкции.
### 59. Чем определяется разрядность поля кода операции в формате команды?
Разрядность кода операции – числом поддерживаемых инструкций.
### 60. Чем определяется разрядность адресного поля в формате команды при использовании только прямой адресации?
Разрядность адресного поля – количеством адресуемых ячеек памяти.
### 61. Назовите методы управления памятью при использовании дискового пространства
Методы управления памятью с диском: страничный, сегментный.
### 62. Назовите методы управления памятью без использования дискового пространства
Методы без диска: статическая и динамическая аллокация.
### 63. Каков порядок выполнения вычислительного процесса в ходе прерываний?
Порядок при прерываниях: сохранение контекста, выполнение обработчика, восстановление контекста.
### 64. Каков порядок доступа к информации в стековой памяти?
В стековой памяти: доступ по принципу LIFO (последним пришёл – первым вышел).
### 65. Что такое ПОЛИЗ?
ПОЛИЗ – форма записи выражений без скобок (постфиксная).
### 66. Каков алгоритм формирования арифметического выражения в ПОЛИЗе?
Алгоритм ПОЛИЗа: последовательный разбор и преобразование выражения.
### 67. Какие конфликты возникают при конвейеризации вычислений?
Конфликты: структурные, по данным, по управлению.
### 68. Назовите основные метрики параллельных вычислений.
Метрики: ускорение, масштабируемость, эффективность.
### 69. Охарактеризуйте закон Амдала.
Закон Амдала – ограничение ускорения вычислений при добавлении процессоров.
### 70. Охарактеризуйте закон Густафсона.
акон Густафсона – параллелизация увеличивает производительность с ростом задачи.
### 71. Охарактеризуйте закон Сана-Ная.
Закон Сана-Ная – компромисс между производительностью и затратами.
